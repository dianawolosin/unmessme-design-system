# UnmessMe Benchmark Test Data

This directory contains the test data for the 4 format benchmark experiments.

## Directory Structure

```
test-data/
├── test-1-json-md-yaml/    # Baseline: JSON tokens + MD specs (YAML frontmatter)
│   ├── components/         # Symlink to packages/design-system/component-specs
│   └── tokens/             # Symlink to packages/design-system/src/tokens
├── test-2-toon-md/         # Toon tokens + MD specs (Toon frontmatter)
│   ├── components/         # Generated by yaml-to-toon.ts
│   └── tokens/             # Generated by json-to-toon.ts
├── test-3-pure-md/         # Pure Markdown (no frontmatter, no JSON)
│   ├── components/         # Generated by strip-frontmatter.ts
│   └── tokens/             # Generated by json-to-markdown.ts
└── test-4-mono-json/       # Monolithic JSON (everything nested)
    ├── components/         # Generated by md-to-monolithic-json.ts OR copied from company
    └── tokens/             # Nested in component JSON
```

## Usage

### Generate Test Data

```bash
# Test 2: Convert to Toon format
npm run transform:test2

# Test 3: Convert to Pure MD
npm run transform:test3

# Test 4: Convert to Monolithic JSON (or copy from company data)
npm run transform:test4
```

### Run Benchmarks

```bash
# Token efficiency test (all formats)
npm run benchmark:tokens

# Chunk inspection (specific format)
npm run benchmark:chunks -- --test=1

# Retrieval precision (all formats)
npm run benchmark:retrieval
```

## Notes

- Test 1 uses symlinks to avoid duplication
- Tests 2-4 require transformation scripts (to be implemented)
- All results are documented in `BENCHMARK_RESULTS.md`
